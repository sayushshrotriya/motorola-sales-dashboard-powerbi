
# 📊 Motorola Mobile Sales Dashboard (Power BI)

A complete interactive Power BI dashboard built to analyze Motorola mobile sales performance across different cities, brands, payment methods, customer ratings, and sales trends.

This project demonstrates data cleaning, data modeling, DAX calculations, and dashboard visualization skills using Power BI.

# 📌 Project Overview

The objective of this project is to transform raw Motorola mobile sales data into meaningful business insights through an interactive dashboard.

The dashboard helps users:

- Monitor overall sales performance
- Analyze product-wise sales
- Compare city-wise sales
- Understand customer ratings
- Identify preferred payment methods
- Track daily and monthly sales trends

---

# 🎯 Problem Statement

Businesses generate huge amounts of sales data every day.

Without proper visualization, it becomes difficult to answer questions like:

- Which city generates the highest sales?
- Which mobile model performs best?
- Which payment method is most used?
- How many transactions happen each month?
- What are customer satisfaction levels?
- Which day has the highest sales?

This dashboard solves these problems by presenting all important KPIs in one place.

# 📈 Dashboard Features

## 1️⃣ KPI Cards

Displays important business metrics:

- Total Sales
- Average Sales
- Total Quantity Sold
- Total Transactions

These KPIs provide a quick overview of overall business performance.

---

## 2️⃣ Monthly Filter (Slicer)

Users can filter the dashboard by month.

Available months:

- January
- February
- March
- April
- May
- June
- July
- August
- September
- October
- November
- December

All visuals update automatically after selecting a month.

---

## 3️⃣ Daily Quantity Trend

Shows the total quantity sold for each day.

Purpose:

- Identify demand fluctuations
- Detect peak sales days
- Observe daily purchasing behavior

Visualization:
- Line Chart

4. Use the month slicer to interact with the dashboard.
5. Hover over charts for detailed insights.
